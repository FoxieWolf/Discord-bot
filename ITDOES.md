# IT-DOES

It is a bit for servers that need a discord bot for $oleplay servers and for people that need help with it.

## What it actualy does

- Session setup
- Infraction setup
- Server backup's

## Commands

- Session

```bash
/startup - Start a session
/setup - Set the session up
/earlyaccess - Send early access 
/release - Release the civilian
/cancel - cancel the session
/end - end the session 
;co - set the co-host
```

- Infraction

```bash
/Infraction - Infract an user
/clearinfractions - Clear all infractions of an user
/editinfraction - Edit the Infraction of an user
/removeinfraction - Remove the Infraction of an user
/staffinfraction - Infract an staff member
/exportinfractions - export all Infractions into an .json file
/infractionstats - See the Stats of all Infraftions
```

- Backup

```bash
/backup - Create a backup of a server.
/backupinfo - Get the infomations of a Backup
/deletebwckup - Delete a backup
/listbackups - Get the list of all Backups
/restore - loads the Full backup of a server
```
