# Discord-bot

## Important
Please have in reminder that you might change a few things before running the bot.

Also Some configuartions are decorations for now since some settings dont work and inside main.js

## Bot instalation 

first copy the bot from my github by doing

```bash
git clone https://github.com/FoxieWolf/Discord-bot.git
cd Discord-bot
```

once you have the full bot you gotta do 

```bash
npm install
```

then you have to configure the bot's settings in config.yml.

once you did that you are ready to go!

## How to get the bot token

- Go to https://discord.com/developers/applications/
- Click on new Application
- Give it a name
- Go into the Bot's Settings
- Click on the Menu symbol/Hambuger Menu
- Click on Bot
- Click on Reset Token
- Copy it and store it somewhere safe
- Now enable ```bash Presence Intent, Server Members Intent, Message Content Intent```
- Then save
